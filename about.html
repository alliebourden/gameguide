About GameGuide

Discover & Play

In 2023 I was accepted into the Get Coding mentorship program through TechNL. This program allows individuals to re-skill into the tech industry by learning the tools and skills needed by current developers. Students are paired with a local mentor to guide them through each module. GameGuide was developed as my module one project using HTML, CSS and Vanilla JavaScript. This application uses API data to allows users to view the top trending games, sort them by name or year, and then view further details on their choosen game.
